# Prototype assets

Drop the hero screenshot here as **`hero-snapshot.png`**:

```
assets/hero-snapshot.png
```

The primary-hero asset panel loads it via `url('assets/hero-snapshot.png')`
(see `.primary-hero--asset-image .primary-hero__asset` in prototype.html),
anchored to the top and cropped at the panel's bottom edge. Toggle it with
the "Hero screenshot" switch in the dev panel.
